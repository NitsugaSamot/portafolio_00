UpTask - Administrador de tareas

Proyecto MERN FullStack

Tecnologías aplicadas:

MongoDB, Express, React, Node, JavasCript,TailwindCSS, HeadlessUI, NodeJS, SocketIo.

Este proyecto es un administrador de tareas.
Incluye registro, confirmación, autenticación, reseteo de contraseñas.
Esta app tiene funcionalidad diferente segun se la quiera ejecutar como administrador de tareas o como colaborador. Como administrador puedes agregar y quitar los colaboradores que necesites y tienes mas permisos que estando como colaborador cuya unica funcion es notificar las tareas completadas.
Tiene interacciones en tiempo real entre administradores y colaboradores de un proyecto.
