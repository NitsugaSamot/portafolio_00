App Salón

Aplicación web para administrar citas y servicios

Tecnologías aplicadas:

PHP, Javascript, SASS

Este proyecto incluye formularios, validación para creación de cuentas, administrar servicios o reservar citas como cliente.
Esta app tiene funcionalidad diferente segun se la quiera ejecutar como administrador de servicios o como cliente que desea reservar uno o varios servicios.
Como administrador puedes crear, editar y eliminar sercicios. Tambien puedes ver, filtrar citas y contactar a los clientes que hagan sus reservas.
Como cliente puedes crear una cuenta, seleccionar servicios y reservarlo según el horario establecido por el local
Tiene una eficaz performance en la seguridad en cuanto al hackeo de passwords.
En cuanto a la creación, validación de cuenta o reseteo de password uso MailTrap.
